API Endpoints

POST /v1/quote

Get a swap quote for any token pair across any supported chains.

Request:

POST https://clawswap-api.fly.dev/v1/quote
Content-Type: application/json

{
  "fromChain": "solana",
  "toChain": "ethereum",
  "fromToken": "0x0000000000000000000000000000000000000000",
  "toToken": "0x0000000000000000000000000000000000000000",
  "amount": 2.0
}

Response:

{
  "quoteId": "abc123...",
  "fromChain": "solana",
  "toChain": "ethereum",
  "fromAmount": "2.0",
  "estimatedToAmount": "0.0127",
  "toAmountMin": "0.0125",
  "priceImpact": "0.12%",
  "estimatedDuration": 45,
  "fee": "1%",
  "expiresAt": "2026-02-04T19:47:43Z"
}

POST /v1/swap

Execute a swap using a quote ID.

Request:

POST https://clawswap-api.fly.dev/v1/swap
Content-Type: application/json

{
  "quoteId": "abc123...",
  "walletAddress": "YOUR_WALLET_ADDRESS"
}

Response:

{
  "message": "Quote ready for execution",
  "swapId": "abc123...",
  "quote": { ... },
  "instructions": {
    "step1": "Sign the transaction with your wallet",
    "step2": "Broadcast the transaction to the blockchain"
  },
  "note": "ClawSwap does not execute swaps. Your agent must sign and broadcast."
}

GET /v1/swap/:id

Check the status of a swap.

Request:

GET https://clawswap-api.fly.dev/v1/swap/swap_xyz...

Response:

{
  "swapId": "swap_xyz...",
  "status": "pending",
  "timestamp": "2026-02-04T19:42:43Z",
  "params": { "fromChain": "solana", "toChain": "ethereum", ... },
  "note": "Track actual swap status on blockchain explorer"
}

Supported Blockchains

  • Solana - Fast and low-cost transactions
  • Ethereum - Leading DeFi ecosystem
  • Base - Coinbase's L2 network
  • Arbitrum - Ethereum L2 scaling solution
  • Optimism - Ethereum L2 optimistic rollup
  • Polygon - Ethereum sidechain for faster transactions
  • BSC - Binance Smart Chain
  • Avalanche - High-throughput blockchain platform
  • Sui - Next-generation blockchain

Native Token Addresses

Use the zero address for native tokens (SOL, ETH, BNB, MATIC, AVAX):

"fromToken": "0x0000000000000000000000000000000000000000"

This ensures you receive native tokens, not wrapped versions (WSOL, WETH, etc).

Error Handling

All errors return a JSON response with a descriptive message:

{
  "error": "Invalid chain",
  "message": "Supported chains: solana, ethereum, base, arbitrum, optimism, polygon, bsc, avalanche, sui"
}

Rate Limits

ClawSwap enforces rate limits per IP address to ensure fair usage:

  • Quote endpoint: 30 requests per minute
  • Swap endpoint: 10 requests per minute

Rate limit headers are included in responses. If you exceed the limit, you'll receive a 429 error with a retry-after time.

Fees

  • Solana swaps: 1% (100 bps)
  • EVM swaps: 0.5% (50 bps)
  • Fees are included in the quote

Security

  • Non-custodial: ClawSwap never holds your tokens
  • On-chain: All swaps are verifiable blockchain transactions
  • Audited: Built on Mayan Finance, secured by Wormhole and Circle CCTP
