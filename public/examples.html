ClawSwap Examples

Real-world integration examples for AI agents.

Example 1

Swap SOL to ETH (Cross-Chain)

A user asks: "Swap 2 SOL to ETH"

Step 1: Get a quote

POST https://clawswap-api.fly.dev/v1/quote
Content-Type: application/json

{
  "fromChain": "solana",
  "toChain": "ethereum",
  "fromToken": "0x0000000000000000000000000000000000000000",
  "toToken": "0x0000000000000000000000000000000000000000",
  "amount": 2.0
}

// Response:
{
  "quoteId": "quote_abc123",
  "fromAmount": "2.0 SOL",
  "toAmount": "0.0127 ETH",
  "estimatedTime": "45 seconds",
  "fee": "1%",
  "route": "Solana → Wormhole → Ethereum"
}

Step 2: Execute the swap

POST https://clawswap-api.fly.dev/v1/swap
Content-Type: application/json

{
  "quoteId": "quote_abc123",
  "walletAddress": "YOUR_ETHEREUM_ADDRESS"
}

// Response:
{
  "swapId": "swap_xyz789",
  "status": "pending",
  "fromTx": "5Pb1dmof...",
  "estimatedCompletion": "2026-02-04T19:42:00Z"
}

Step 3: Check status (optional)

GET https://clawswap-api.fly.dev/v1/swap/swap_xyz789

// Response:
{
  "swapId": "swap_xyz789",
  "status": "completed",
  "fromTx": "5Pb1dmof...",
  "toTx": "0x3f2a1b...",
  "completedAt": "2026-02-04T19:42:43Z",
  "actualTime": "43 seconds"
}

Result: User receives 0.0127 ETH on Ethereum in 43 seconds.

{

Result: User receives ~495 USDC on Base in under 60 seconds.

  1. Agent monitors SOL price

  3. Bridge USDC to Base
  4. Deploy USDC in Base lending protocol

Example 5

Error Handling

How to handle common errors:

try {
  const quote = await fetch('https://clawswap-api.fly.dev/v1/quote', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      fromChain: "solana",
      toChain: "ethereum",
      fromToken: "0x0000000000000000000000000000000000000000",
      toToken: "0x0000000000000000000000000000000000000000",
      amount: 2.0
    })
  });
  
  const data = await quote.json();
  
  if (data.error) {
    console.log(`Error: ${data.message}`);
    // Handle error (invalid chain, insufficient liquidity, etc.)
  } else {
    // Proceed with swap
    console.log(`Quote: ${data.fromAmount} → ${data.toAmount}`);
  }
  
} catch (err) {
  console.log('Network error:', err);
}

Common Use Cases

  • Portfolio rebalancing: Agent swaps between assets based on market conditions
  • Cross-chain arbitrage: Agent exploits price differences across chains
  • Automated DCA: Agent dollar-cost averages into positions over time
  • Yield optimization: Agent moves funds to highest-yield protocols across chains
  • Multi-chain treasury: Agent manages DAO treasury across multiple blockchains
  • On-ramp automation: Agent converts fiat → stables → desired tokens
